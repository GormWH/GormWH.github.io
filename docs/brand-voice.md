# Brand voice

Applies when writing copy on the site (hero, about, project descriptions, blog, etc.).

## Voice

- **Personal copy:** `I` voice — calm and direct (no `we`).
- **Project descriptions:** neutral third-person.

## Typography rules

- **No emoji.**
- Allowed glyphs: `→ · — ※ ✻`.
- Sentence case for headings.
- ALL CAPS only for tiny mono labels.

## Languages in scope

EN / JP / KR. The live hero already mixes all three:

```
한국어 · 日本語 · English
```
