# AGENTS.md — GormWH Portfolio

> This file is a map, not an encyclopedia.
> It explains what this repository is and where to find details.

## Project Identity

- **What**: Personal portfolio website for SuHong Park
- **Goal**: Present profile, system-thinking, and projects as proof of work
- **Stack**: Astro 6, Tailwind CSS 4, TypeScript
- **Hosting**: GitHub Pages via GitHub Actions (`withastro/action`)
- **URL**: https://gormwh.github.io

## Architecture (quick summary)

- Route pages are in `src/pages/` and currently render directly with `BaseLayout`.
- Shared shell/navigation lives in `src/layouts/BaseLayout.astro` and `src/components/`.
- Project entries are markdown files in `projects/`.
- The `projects` content collection schema is defined in `src/content.config.ts`.
- Dynamic project pages are generated by `src/pages/projects/[...slug].astro`.

## Objectives and Constraints

1. Keep this a static site (no server runtime, no database, no auth layer).
2. Keep dependency surface small unless there is a clear benefit.
3. Treat project writeups as the core portfolio evidence.
4. Prefer clarity and maintainability over visual/technical complexity.
5. Keep documentation truthful to current implementation.
